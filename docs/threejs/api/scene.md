# 场景 scene

## createScene 创建场景

创建场景和灯光

## setScene 设置场景

**Attributes**

| 参数  | 说明                 | 类型     | 可选值 | 默认值   | 备注             |
| :---- | :------------------- | :------- | :----- | :------- | :--------------- |
| value | 从模型读取的场景实体 | Object3D | -      | -        |                  |
| type  | 要设置的场景类型     | String   | -      | ''/'mul' | 判断是否是多厂区 |
