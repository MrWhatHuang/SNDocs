# 其它

## setObjectVisible 控制物体显隐

**Attributes**

| 参数    | 说明             | 类型           | 可选值     | 默认值 | 备注 |
| :------ | :--------------- | :------------- | :--------- | :----- | :--- |
| object  | 物体或物体名集合 | Object3D/Array | -          | -      |      |
| visible | 物体显隐性       | Boolean        | true/false | true   |      |

## createGroup 创建组

**Attributes**

| 参数 | 说明   | 类型   | 可选值 | 默认值 | 备注 |
| :--- | :----- | :----- | :----- | :----- | :--- |
| name | 组名称 | String | -      | -      | ''   |
