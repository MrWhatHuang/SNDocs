# 物体透明 opacity

## setOpacity 设置透明

**Attributes**

| 参数   | 说明         | 类型                | 可选值     | 默认值   | 备注 |
| :----- | :----------- | :------------------ | :--------- | :------- | :--- |
| object | 物体         | Object3D/Group/Mesh | -          | -        |      |
| op     | 透明度       | Float               | -          | 0.4      |      |
| col    | 颜色         | Number              | -          | 0x003c9a |      |
| ignore | 是否忽略楼板 | Boolean             | true/false | true     |      |

## removeOpacity 移除透明

**Attributes**

| 参数   | 说明 | 类型                | 可选值 | 默认值 | 备注 |
| :----- | :--- | :------------------ | :----- | :----- | :--- |
| object | 物体 | Object3D/Group/Mesh | -      | -      |      |
