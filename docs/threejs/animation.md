# 光晕动画 Animation

## initAnimation 绘制光晕动画

### Attributes

| 参数  | 说明     | 类型     | 可选值 | 默认值    | 备注                                           |
| :---- | :------- | :------- | :----- | :-------- | :--------------------------------------------- |
| obj   | 附着物体 | Object3D | -      | -         | 附着目标物体                                   |
| color | 颜色     | String   | -      | '#f60000' | 第一次传入后轨迹数据会缓存到内存中直至移除轨迹 |

## removeAnimation 移除光晕动画

### Attributes

| 参数 | 说明     | 类型     | 可选值 | 默认值 | 备注         |
| :--- | :------- | :------- | :----- | :----- | :----------- |
| obj  | 附着物体 | Object3D | -      | -      | 附着目标物体 |
